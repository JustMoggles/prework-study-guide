HTML

  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user
  • The footer element contains information at the bottom of the page
  • Semantic elements and alt attributes are important for accesibility
  • There are six sizes of header elements
  • Child elements are elements which are nested together within a parent element

CSS

  • A margin indicates how much space we want around the outside of an element
  • A padding indicates how much space we want around the content inside an element
  • DNRY code is short for "Do Not Repeat Yourself code" - a method for keeping code cleaner and simpler
  • Using External CSS Style Sheets are the most common way to style a webpage
  • Separation of concerns is a key design principle, where each section of code is split into different files according to their use
  • A declaration contains to important componenets: the property we want to apply to and the value of that property

Git

  • Git is the most commonly used version control
  • Git uses the command line
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • We can create different branches in our workflow through Git
  • git log: shows all the commits in the current branch's history

JavaScript

  • A variable is a named container that allows us to store data in our code
  • Control flow is the order in which a computer executes code in a script
  • You can comment out code without deleting it by putting /* before and */ after the comment
  • An array is a single variable that is used to hold a group of data and is zero indexed
  • We can introduce decision making to our code by using condition statements
  • A function is a set of instructions which direct your computer on how to perform a task
